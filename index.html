TL;DR

Six intentions, six orders

Pick the intention first. The order type follows.

Intention Order
Punching through, decision is madeMarket
Patient at a levelLimit + GTC
Now or never, no restingLimit + IOC
Maker-only, and I want to actually fillLimit + TOB ✦
Wake me up if I'm wrongStop Market
Wake me up, but cap the slippageStop Limit

✦ TOB is Pacifica-exclusive — post-only orders that reposition instead of getting cancelled when the book moves.

def choose_order(side, size_in_usd, conviction, market):
    if conviction > 0.8 and size_fits_book:
        return market_order()              # punch through
    if conviction > 0.5:
        return limit(tif="TOB")            # Pacifica exclusive
    return limit(tif="GTC")                # patient

# 3. Open the position AND attach a stop, in one call
def safe_open(side, size_in_usd, conviction, stop_pct=2.0):
    market = read_market()
    order  = choose_order(side, size_in_usd, conviction, market)
    entry  = place_entry(order)
    if entry.filled:
        place_stop_market(reduce_only=True)  # never optional
    return entry
